# Anay Parwal Portfolio

A polished static portfolio for Anay Parwal, built for GitHub Pages and focused on computer science, data science, cybersecurity, athletics, and internship readiness.

## Local Preview

```sh
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## GitHub Pages

Recommended repository:

```text
https://github.com/gluepixel/gluepixel.github.io
```

When pushed to `main`, the included workflow at `.github/workflows/pages.yml` deploys the static site to:

```text
https://gluepixel.github.io
```

In GitHub repository settings, set Pages to deploy from GitHub Actions.
